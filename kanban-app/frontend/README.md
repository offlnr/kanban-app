# KanbanApp — Frontend

React + TypeScript frontend for the KanbanApp project management tool.

## Tech

- React 19 + TypeScript
- Tailwind CSS v4
- @dnd-kit (drag and drop)
- react-i18next (EN/ES)
- React Router v7
- Axios

## Development

```bash
npm install
npm run dev
```

Requires the backend running on `http://localhost:8000`.
